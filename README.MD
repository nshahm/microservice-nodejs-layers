# Microservice template for nodejs 
* Base template for microservice using node js with below packages

## Packages list

NodeJS

Packages

* HTTP
    ** express
    
* Logging
    ** winston
    ** morgan    
    
* Lint
    ** eslint
    
* Code  
    ** typescript (typed superset of javascript)

* database
    ** mongodb
    ** ODM 
        *** mangoose

* testing
    ** mocha (framework)
    ** chai (asserting library)
    
* Build
    webpack
    gulp (if needed)
    

npm run build 
node ./dist/server.js
